### :dart: My Projects:

- 📱 Non ci resta che Guardare

  - Android app to note down movies and Tv Series to watch
  - Project developed for the "Laboratorio di programmazione per sistemi mobili e tablet" course at UniTN, and later continued as a personal project.
  - [Link to repo](https://github.com/bettozzo/NonCiRestaCheGuardare)
  - [Link to website version](https://github.com/bettozzo/nrg)

- 🦾 ClawMachine

  - A claw machine developed with embedded C.
  - Project developed in collaboration with 3 fellow students for the "Embedded Software for the Internet Of Things" course at UniTN.
  - [Link to repo](https://github.com/Clown-Machine/ClawMachine)

- 🔀Spotify Shuffler

  - Uses Python to fetch data from the Spotify API and creates a new (randomly) shuffled playlist
  - [Link to repo](https://github.com/bettozzo/SpotifyShuffler)

- 🌍 Tum4World - Website

  - The front-end uses jsp, HTML, CSS and pure JavaScript.
  - The back-end uses Java and Apache-Derby.
  - Data is transfered thanks to Servlets.
  - Project developed in collaboration with 3 fellow students for the "Introduzione alla Programmazione Web" course at UniTN
  - [Link to repo](https://github.com/bettozzo/ProgettoWebUniTN)
